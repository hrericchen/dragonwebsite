# Execution Scripts

This folder contains **deterministic Python scripts** that handle:
- API calls
- Data processing
- File operations
- Database interactions

## Guidelines
- Scripts should be **reliable, testable, and well-commented**.
- Each script should do one thing well.
- Use environment variables from `.env` for credentials — never hardcode secrets.
- Before writing a new script, check if one already exists that does what you need.
